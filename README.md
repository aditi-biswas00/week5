# Customer Sales Analysis Dashboard

## Objective
Analyze customer purchasing patterns, identify top customers, and create a comprehensive sales performance dashboard using Python and Pandas.

## Features
- Customer Lifetime Value Analysis
- Regional Sales Distribution
- Monthly Sales Trend Analysis
- Product Performance Analysis
- Pivot Table Reporting
- Professional Dashboard Visualizations

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Jupyter Notebook

## Run Project
```bash
pip install -r requirements.txt
jupyter notebook
```
